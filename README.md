# GTIN
 Productos fabricados por una empresa del sector de la alimentación.
 Cada unidad de consumo de producto que adquiere un usuario final tiene un código GTIN-13 (antes
 EAN-13) que identifica el tipo de producto. Consta de 13 dígitos numéricos.
 Para productos con dimensiones pequeñas se utiliza GTIN-8, con 8 dígitos. En Estados Unidos y
 Canadá se utiliza UPC-A, similar a GTIN-13, pero con 12 dígitos.
 En GTIN-13 se indica para cada tipo de producto un código de país (84 = España), un identificador de
 empresa (de 5 a 8 dígitos numéricos) registrado en un organismo nacional encargado de la aplicación
 de los estándares GS1 y un identificador de tipo de producto (de 2 a 5 dígitos, dependiendo de la
 longitud del código de empresa), finalizando con un dígito de control.
 Se utiliza también GTIN-14 (antes EAN-14) para asignar un código a un tipo de contenedor
 (normalmente una caja) que embala un cierto número de unidades de un tipo de producto.
 Consta de 14 dígitos: una clave de un dígito
 (de 1 a 8, no se utiliza el 0 ni el 9) que indica
 el número de unidades embaladas, más los
 12 primeros dígitos del código GTIN-13 del
 producto, finalizando con un dígito de control
 que se calcula como en GTIN-13 salvo que
 hay una posición más.
